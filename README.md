# Artificial-neural-network
This project focuses on building an Artificial Neural Network (ANN) for breast cancer classification. The model is trained on the Breast Cancer dataset to predict whether a tumor is malignant or benign based on various input features.

Key Highlights:
- Data preprocessing using normalization/standardization
- Model built using ANN with hidden layers and activation functions
- Training using backpropagation and optimization techniques
- Evaluation using accuracy and loss metrics

Technologies Used:
- Python
- NumPy, Pandas
- Scikit-learn
- TensorFlow / Keras
